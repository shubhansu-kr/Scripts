# git-boost

`git-boost` is a Zsh utility to:

- Bootstrap a new GitHub repo with standard files
- Link and sync local repos to GitHub
- Standardize all or specific local repos with a README, LICENSE, and CONTRIBUTING.md

### CLI

```bash
